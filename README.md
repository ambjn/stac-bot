# STAC - Smart Settlement Tool

A Telegram bot that manages poker game rooms by tracking player buy-ins, chip counts, and automatically calculating profit/loss settlements between players.

## Use Case

Host creates a home poker game room, invites friends through Telegram links, everyone logs their buy-ins during the game, records final chip amounts at the end, and the bot calculates the optimal settlement (minimizing transactions) so players know exactly who pays whom.
